# ✨ ميزة اختيار الشعار المتعدد

## 📋 ما تم إضافته:

### 1. **مكتبة شعارات جاهزة**
- 4 شعارات جاهزة في مجلد `public/logooo/`
- يمكن إضافة المزيد بسهولة
- تظهر كخيارات قابلة للاختيار في المحرر

### 2. **واجهة اختيار الشعار**
في تبويب **المحتوى** → قسم **الشعار**:

```
┌─────────────────────────────────────┐
│  الشعار                             │
├─────────────────────────────────────┤
│  [شعار 1] [شعار 2] [شعار 3] [شعار 4] │  ← اختيار من متعدد
│                                     │
│  [❌ إزالة الشعار]                  │  ← زر الإزالة
│                                     │
│  ───────── أو ─────────             │
│                                     │
│  [📁 رفع شعار مخصص]                │  ← رفع ملف جديد
│  [📁 رفع صورة]                     │
└─────────────────────────────────────┘
```

### 3. **زر إزالة الشعار**
- يظهر فقط عند وجود شعار مختار
- يزيل الشعار تماماً من الشريحة
- لون أحمر للتمييز

---

## 🎨 كيفية الاستخدام:

### اختيار شعار من المكتبة:
1. افتح المحرر
2. اذهب لتبويب **المحتوى**
3. في قسم **الشعار**، اضغط على أي شعار
4. سيظهر ✅ على الشعار المختار
5. سيتم عرض الشعار فوراً في المعاينة

### إزالة الشعار تماماً:
1. اضغط على زر **❌ إزالة الشعار**
2. سيختفي الشعار من الشريحة

### رفع شعار مخصص:
1. اضغط على **رفع شعار مخصص**
2. اختر ملف من جهازك
3. سيتم استخدامه مباشرة

## 🔧 إضافة شعارات جديدة:

### الخطوة 1: أضف الملف
ضع ملف الشعار في `public/logooo/`

```bash
# مثال
copy my-logo.png public/logooo/
```

### الخطوة 2: سجل الشعار
في `constants.ts`، أضف للشعارات:

```typescript
export const LOGO_OPTIONS = [
  // ... الشعارات الحالية
  { id: 'logo-5', name: 'شعار 5', path: '/logooo/logo-5.png' },
];
```

### الخطوة 3: جاهز!
الشعار سيظهر تلقائياً في المحرر

## 🆘 حل المشاكل:

### الشعار لا يظهر؟
- تحقق من مسار الملف في `LOGO_OPTIONS`
- تأكد أن الملف موجود في `public/logooo/`
- جرب تحديث الصفحة (Ctrl+F5)

### الشعار مشوه؟
- استخدم شعارات مربعة (1:1)
- تأكد من دقة كافية (200x200 على الأقل)

### لا يمكن إزالة الشعار؟
- تأكد أن هناك شعار مختار
- جرب تحديث الصفحة
