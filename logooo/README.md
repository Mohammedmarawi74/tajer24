# 📍 إضافة شعاراتك الخاصة

## كيفية إضافة شعارات جديدة:

### 1. أضف ملفات الشعار إلى مجلد `public/logooo/`
- قم بنسخ ملفات الصور (PNG, JPG, SVG) إلى المجلد
- يفضل استخدام صيغة PNG مع خلفية شفافة
- الحجم الموصى به: 200x200 بكسل

### 2. سجل الشعارات في `constants.ts`

أضف الشعار الجديد إلى مصفوفة `LOGO_OPTIONS`:

```typescript
export const LOGO_OPTIONS = [
  { id: 'logo-1', name: 'شعار 1', path: '/logooo/logo-1.png' },
  { id: 'logo-2', name: 'شعار 2', path: '/logooo/logo-2.png' },
  { id: 'logo-3', name: 'شعار 3', path: '/logooo/logo-3.png' },
  { id: 'logo-4', name: 'شعار 4', path: '/logooo/logo-4.png' },
  { id: 'logo-5', name: 'شعار 5', path: '/logooo/logo-5.png' }, // الجديد
];
```

### 3. إعادة تشغيل خادم التطوير

```bash
npm run dev
```

## 📝 ملاحظات مهمة:

- ✅ المسارات يجب أن تبدأ بـ `/logooo/`
- ✅ أسماء الملفات يجب أن تطابق تماماً (case-sensitive)
- ✅ عند النشر على Vercel، تأكد من رفع مجلد `public` كاملاً
- ✅ يمكن استخدام أي عدد من الشعارات

## 🎨 أفضل الممارسات:

1. **الحجم**: استخدم شعارات بحجم 200x200 بكسل أو متقاربة
2. **الصيغة**: PNG مع خلفية شفافة للحصول على أفضل مظهر
3. **التناسق**: حاول جعل جميع الشعارات بنفس الحجم التقريبي
4. **الاسم**: استخدم أسماء وصفية للشعارات (شعار-الشركة، شعار-الجهة، إلخ)

## 🔄 إزالة شعار:

1. احذف الشعار من مجلد `public/logooo/`
2. احذفه من مصفوفة `LOGO_OPTIONS` في `constants.ts`
3. أعد تشغيل خادم التطوير

---

**💡 نصيحة:** يمكن للمستخدمين أيضاً رفع شعارات مخصصة مباشرة من المحرر دون إضافتها للمكتبة الدائمة.
